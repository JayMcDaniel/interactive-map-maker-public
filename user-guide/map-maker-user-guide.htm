About the Map Maker

The Interactive Map Maker was developed to create 508 compliant data visualizations for the Visualization Gallery on the DOL Open Data Portal. The Office of Data Governance invites agencies to use this tool to create visualizations for the Visualization Gallery, internal analysis, or for your webpages.
